# cologne-hugo

_Created: 14-06-2026 · Last updated: 11-07-2026_

CDSL **build-meta** repository in the [Sanskrit Lexicon](https://github.com/sanskrit-lexicon) project,
intended to host static sites generated by [Hugo](https://gohugo.io/) for the
Cologne Digital Sanskrit Dictionaries. It is **experimental** and not part of the
production pipeline.

## Status

This is currently a **scaffold**: the repository holds project metadata and CI
configuration only — there is no Hugo site content, theme, or build workflow
committed yet. What exists today:

- [`LICENSE`](https://github.com/sanskrit-lexicon/cologne-hugo/blob/main/LICENSE) — GNU General Public License v3.0.
- [`CLAUDE.md`](https://github.com/sanskrit-lexicon/cologne-hugo/blob/main/CLAUDE.md) — agent guidance for this repo.
- [`.github/dependabot.yml`](https://github.com/sanskrit-lexicon/cologne-hugo/blob/main/.github/dependabot.yml) and
  [`.github/workflows/dependabot-auto-merge.yml`](https://github.com/sanskrit-lexicon/cologne-hugo/blob/main/.github/workflows/dependabot-auto-merge.yml)
  — GitHub Actions dependency updates with hands-off auto-merge.

When a Hugo site is added, its build/publish workflow will live under
[`.github/workflows/`](https://github.com/sanskrit-lexicon/cologne-hugo/tree/main/.github/workflows).
